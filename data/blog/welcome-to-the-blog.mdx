This blog is my public notebook.

I wanted a place that stays fast, simple, and easy to keep up to date. The goal is not to publish long, polished essays every time. It is to capture useful context while it is still fresh.

Expect a mix of project write-ups, short technical notes, and occasional reflections on how I work. If a post can save me time the next time I run into the same problem, it belongs here.

## What I care about

- Keeping the writing process lightweight.
- Making posts easy to scan months later.
- Choosing defaults that make the site easy to maintain.

That is enough for a first post. The rest should come from shipping.
